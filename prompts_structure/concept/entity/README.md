# Entity (Non-Humanoid Sentient Entity)

## What This Is
Records the concept design prompt structure for **sentient non-humanoid entities** — AI manifestations, god-projections, alien consciousness, swarm beings, abstract rule-keepers, etc. Used to generate multi-panel reference sheets containing: default form main visual, viewing angles, detail close-ups, form variations, and emotion states.

## Available Model Variants
- `general.md` — compatible with Gemini 2.5 Flash Image and GPT image models (default)
- (Future: Midjourney, DALL-E, Flux, etc.)

## Usage
- If the user specifies a model name, load the corresponding variant file from this directory
- If no model is specified, default to `general.md`
- For image layout structure, see `concept-sheet/entity-sheet/`
