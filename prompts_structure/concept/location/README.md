# Location (Environment / Setting)

## What This Is
Records the concept design prompt structure for **story locations and environments** — city streets, interiors, landscapes, fantasy worlds, etc. Used to generate multi-panel reference sheets containing: establishing shots, alternate angles, top-down overview, mid-distance structure showcases, architectural detail close-ups, and a color palette reference.

## Available Model Variants
- `general.md` — compatible with Gemini 2.5 Flash Image and GPT image models (default)
- (Future: Midjourney, DALL-E, Flux, etc.)

## Usage
- If the user specifies a model name, load the corresponding variant file from this directory
- If no model is specified, default to `general.md`
- For image layout structure, see `concept-sheet/location-sheet/`
