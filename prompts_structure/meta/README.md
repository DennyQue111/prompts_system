# Meta — Cross-Cutting Rules

## What This Is
Shared reference files that apply across all prompt types. These are not prompt templates — they are constraints, methodology, and hygiene rules to consult before writing any prompt.

## Available Files

| File | What It Is | When to Read |
|------|-------------|--------------|
| `prompt-hygiene.md` | Universal anti-pollution rules for all prompt writing | Before writing any prompt |
| `gpt-image-hygiene.md` | GPT-specific anti-noise methodology — material-light definitions, clean rendering blocks, dangerous word replacements | **Mandatory** before writing any GPT prompt |

## Usage
- `prompt-hygiene.md` → always consult, regardless of model
- `gpt-image-hygiene.md` → read before writing any prompt targeting GPT image models
