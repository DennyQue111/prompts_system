# World View (Visual Constitution)

## What This Is
Records the prompt structure for generating **world-view establishing shots** — the visual constitution that defines the project's look across 9 dimensions (inhabitants, architecture, landscape, daily life, etc.) before any character or location is designed. Based on the MOKEAIGC 9-Aspect World-Building framework.

## Available Files

| File | What It Is |
|------|-------------|
| `midjourney_animation.md` | World-view shots for Midjourney — animation/CG anime visual direction |
| `midjourney_realistic.md` | World-view shots for Midjourney — photorealistic/realistic visual direction |

## Usage
- Choose the variant based on the project's visual direction (animation vs. realistic)
- These are Midjourney-only files — for Gemini/GPT world-view generation, adapt the aspect descriptions from these files
- World-view should be established BEFORE any character or location concept work — it sets the visual DNA everything else inherits
